1. Описание проекта
Этот проект - 2D платформер с несколькими уровнями, со сбором монет,
с глобальным рейтингом всех игроков (но это не точно:)).

2. Возможности персонажа.
Умеет ходить, прыгать, бегать, собирать монетки

3. Хранение данных
Приложение будет хранить все рекорды в таблицах и базе данных.

4. Модули, которые будут использоваться.
pygame, sys, mysql-connector-python
